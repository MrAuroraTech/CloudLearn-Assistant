# 1 简介

云学伴 - 智能刷课助手是一款专为传智播客高校学习平台（ityxb.com / ithb.com）设计的油猴脚本，支持自动刷课、智能答题、多题库搜索、AI 搜题等功能。脚本采用 Tampermonkey（篡改猴）运行，无需安装任何额外软件。

# 2 功能总览

| **功能模块** | **说明**                                        |
| ------------ | ----------------------------------------------- |
| 自动刷课     | 按顺序自动播放视频，完成后自动跳转下一节        |
| 自定义倍速   | 支持  1x ~ 16x 倍速，实时滑块调节               |
| 智能防卡顿   | 监控缓冲区动态调速，严重卡顿自动恢复            |
| 多题库搜题   | 言溪题库  / AI API / 百度  / 必应四种渠道       |
| 智能答题     | 自动识别题型，搜题后自动填入并提交              |
| 可拖拽面板   | 面板支持自由拖拽移动，可最小化                  |
| SPA 支持     | 单页应用路由变化自动重新初始化                  |
| 配置持久化   | 题库  Token、AI  Key 等配置自动保存，刷新不丢失 |

## 3.2 安装脚本

1. 打开 Tampermonkey 管理面板（点击浏览器右上角篡改猴图标）
2. 点击 “+” 新建脚本
3. 清空编辑器中所有内容，粘贴脚本代码
4. 按 Ctrl+S 保存
5. 打开传智播客学习平台即可自动运行

## 3.3 Edge 浏览器特殊设置

如果使用 Edge 浏览器，需要额外开启开发者模式：

6. 地址栏输入 edge://extensions/
7. 左下角打开“开发人员模式”开关，重启浏览器

# 5 题库配置详解

## 5.1 言溪题库

言溪题库是第一优先级搜题渠道，命中率达 98% 以上。配置方法：

8. 访问 https://tk.enncy.cn 并登录
9. 进入个人中心，点击“复制题库配置”
10. 在脚本配置 Tab 的 Token 输入框中粘贴完整配置
11. 打开“启用题库”开关
12. 点击“测试题库连接”确认可用

**支持的配置格式：**

•    完整 JSON 配置（推荐）：直接粘贴言溪题库复制的配置，脚本自动提取 token、url、handler 等字段

•    纯 Token：直接输入 token 字符串（如 fab84c7b14114d89bf16eac475f3c512）

## 5.2 AI 搜题

支持三个 AI 提供商，配置方法如下：

| **提供商** | **API** **地址**          | **获取**  **API Key**              |
| ---------- | ------------------------- | ---------------------------------- |
| 智谱清言   | open.bigmodel.cn          | https://open.bigmodel.cn           |
| DeepSeek   | api.deepseek.com          | https://platform.deepseek.com      |
| 豆包       | ark.cn-beijing.volces.com | https://console.volcengine.com/ark |

配置步骤：

13. 注册对应平台并获取 API Key
14. 在配置 Tab 中选择提供商，输入 API Key
15. 打开“启用 AI”开关，点击“测试 AI 连接”确认

# 6 搜题逻辑

脚本按照以下固定顺序搜题，前一个渠道找到答案后不再尝试后续渠道：

| **优先级** | **渠道** | **说明**                | **置信度** |
| ---------- | -------- | ----------------------- | ---------- |
| 1          | 言溪题库 | 专业题库，命中率  98%+  | 95%        |
| 2          | AI API   | 智谱  / DeepSeek / 豆包 | 85%        |
| 3          | 百度搜索 | 网页搜索兆底            | 变动       |
| 4          | 必应搜索 | 网页搜索兆底            | 变动       |

# 7 支持的题型

| **题型** | **识别方式**               | **填入策略**                           |
| -------- | -------------------------- | -------------------------------------- |
| 单选题   | 检测  .el-radio 元素       | 字母匹配  / 文字匹配  / 模糊匹配       |
| 多选题   | 检测  .el-checkbox 元素    | 解析字母  + 文字分别点击               |
| 判断题   | 检测  .tf-content 或关键词 | 匹配“正确/错误”关键词                  |
| 填空题   | 检测  input/textarea 元素  | Vue 兼容写入（nativeInputValueSetter） |

# 8 智能防卡顿机制

脚本内置 AntiBuffer v2 模块，通过监控视频前方缓冲区动态调节倍速：

| **缓冲状态**      | **操作**                     |
| ----------------- | ---------------------------- |
| 前方缓冲  < 3 秒  | 立即大幅降速（降速步长  x2） |
| 前方缓冲  < 10 秒 | 正常降速（每次降  2x）       |
| 前方缓冲  > 30 秒 | 平滑提速（每次升  2x）       |
| 缓冲适中          | 保持当前倍速                 |
| 卡顿超过  3 秒    | 大幅降速  + seek 触发预加载  |

# 9 常见问题

## Q1: 脚本不运行，显示“此脚本还未被执行”

**解决方法：**确保已安装最新版本脚本。Edge 浏览器需开启开发者模式（edge://extensions/）。刷新页面后查看 Tampermonkey 图标是否显示数字 1。

## Q2: 题库显示“无效的用户凭证”

**解决方法：**Token 已过期，需要重新去言溪题库复制新的配置。复制后直接粘贴到 Token 输入框即可，脚本会自动解析 JSON 配置。

## Q3: 答题后倍速变成 1x

**解决方法：**这是平台的正常行为——从习题页跳转回视频时，新视频元素默认 1x。脚本已内置自动恢复倍速的逻辑，会在跳转后 2 秒自动恢复目标倍速。

## Q4: 视频播放卡顿

**解决方法：**脚本内置智能防卡顿模块，会自动根据网络状况降低倍速。如果仍然卡顿，可尝试手动将倍速调低到 8x 或 4x。

## Q5: 面板挡住页面内容

**解决方法：**按住面板头部标题栏可以拖拽移动面板位置。点击最小化按钮可将面板收缩为圆形图标。
